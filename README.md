# ZGIRC时间同步

## 简介

ZGIRC时间同步工具，支持图形界面、开机自启动和定时同步。

## 功能特性

- **图形界面**：简洁直观的GUI设计
- **时间同步**：使用阿里云NTP服务器同步时间
- **开机自启动**：支持开机自动运行时间同步程序
- **定时同步**：支持设置自动同步间隔
- **日志管理**：支持保存日志、清理过期日志
- **管理员权限**：自动处理系统时间修改

## 系统要求

- 操作系统：Windows 7/8/10/11
- Python版本：3.8及以上（打包版无需Python）

## 文件说明

```
发布版/
├── ZGIRC_TimeSync.exe    # 主程序（GUI配置界面）
└── data/
    ├── time_sync.exe     # 时间同步程序
    └── config.json       # 配置文件
```

## 使用方法

### 快速开始

1. 双击运行 `ZGIRC_TimeSync.exe`
2. 配置参数：
   - 时区偏移：填写小时数（如中国时区填8）
   - 同步间隔：填写分钟数
   - 日志保留：填写天数
3. 点击"保存配置"
4. 勾选"开机自动启动"实现开机自动同步
5. 勾选"定时自动同步"实现定时自动同步
6. 点击"立即同步时间"立即同步一次时间

### 配置文件说明

配置文件位于 `data/config.json`：

```json
{
    "ntp_servers": [
        "time1.aliyun.com",
        "time2.aliyun.com",
        "time3.aliyun.com",
        "time4.aliyun.com",
        "pool.ntp.org"
    ],
    "time_offset": 8,
    "retry_count": 3,
    "sync_interval": 3600,
    "log_days": 7
}
```

- `ntp_servers`：NTP服务器列表
- `time_offset`：时区偏移（小时）
- `retry_count`：重试次数
- `sync_interval`：同步间隔（秒）
- `log_days`：日志保留天数

### 开机自启动

- 勾选"开机自动启动"后，开机时会自动运行时间同步程序
- 时间同步程序会在后台静默运行，自动同步时间

### 日志位置

日志文件位于程序同目录下的 `log` 文件夹中。

## 注意事项

1. 修改系统时间需要管理员权限，程序会自动请求管理员权限
2. 建议开启开机自启动，确保时间始终准确
3. 如遇到同步失败，请检查网络连接或尝试更换NTP服务器
